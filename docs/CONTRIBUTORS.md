# Contributors

This project depends on rapid public-response engineering, data processing, and field-oriented UX work. Credits here are kept explicit so operational improvements remain attributable even when follow-up commits adapt or reimplement parts of an earlier pull request.

## Mobile PWA And Offline Workflow

Enderson Omaña (`@endersonO`) contributed the original installable mobile PWA and offline-access work in [endersonO/crisis-damage-intelligence#1](https://github.com/endersonO/crisis-damage-intelligence/pull/1).

That work covered the mobile install path, web app manifest/icons, offline route, service worker strategy, install prompt behavior, language persistence, mobile map controls, and mobile/tablet verification. A later upstream PR shipped a reimplemented version of this surface without preserving Enderson's commit authorship. This credit records the contribution explicitly and should be preserved in future release notes or public handoffs that mention the PWA/mobile offline work.
